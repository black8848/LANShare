# FileT - 内网文件传输工具

一个简单的内网文件和文字传输工具，支持在局域网内快速分享文件和文字。

## 功能

- 文件上传/下载（支持拖拽，多文件）
- 文字分享（支持复制到剪贴板）
- 响应式界面，支持手机访问
- 无需注册，开箱即用

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
python app.py
```

启动后访问：
- 本机：http://localhost:8080
- 内网：http://你的IP:8080

## 配置

在 `app.py` 中可修改：

| 配置项 | 默认值 | 说明 |
|-------|-------|------|
| `port` | 8080 | 服务端口 |
| `MAX_CONTENT_LENGTH` | 500MB | 最大上传大小 |
